▷ 내작 객체
▶ String : 문자열을 다룸 
	속성
		length -> 길이 반환
	메소드
		includes('문자열') -> 인자로 전달된 문자열이 포함되어있다면 true
		slice(시작, 종료직전) -> 지정된 범위 추출
		split('구분자') -> 문자열을 배열로 분할 후 배열 반환
		trim() -> 문자열의 양 끝 공백 제거 후 반환 

▶ Array : 배열을 다룸
	속성
		length -> 길이 반환
	메소드
		push(요소) -> 마지막 요소 삭제
		pop() -> 마지막 요소 삭제
		forEach(함수) -> 배열의 각 요소마다 함수 호출
